## Setup & Instalasi

### Clone Project
```bash
git clone https://github.com/zuLmeister/golang-jwt.git
cd golang-jwt

Install Dependency
go mod tidy

Buat Database MySQL
CREATE DATABASE myappdb;

Generate JWT Secret Key ðŸ”‘

Windows (PowerShell):

[Convert]::ToBase64String((1..64 | ForEach-Object {Get-Random -Maximum 256}))


Linux / macOS:

openssl rand -base64 64

Buat file .env

Isi sebagai berikut:

DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASS=yourpassword
DB_NAME=myappdb

ACCESS_SECRET=isi_dengan_secret_generated
REFRESH_SECRET=isi_dengan_secret_generated

Jalankan aplikasi
go run main.go


Server default:
http://localhost:8080

API Endpoint
Method	Endpoint	Deskripsi
POST	/register	Registrasi user
POST	/login	Login user & mendapatkan token
POST	/refresh	Refresh access token dengan refresh token
GET	/profile	Protected route
Contoh Request

Register
POST /register
Content-Type: application/json

{
  "name": "Zul",
  "email": "zul@example.com",
  "password": "123456"
}

Login
POST /login
Content-Type: application/json

{
  "email": "zul@example.com",
  "password": "123456"
}


Response:

{
  "access_token": "xxxxx",
  "refresh_token": "xxxxx"
}

Protected Route
GET /profile
Authorization: Bearer <access_token>
